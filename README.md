# 🏫 College Website Replica

A fully responsive, multi-page **College Website Replica** built using pure **HTML5** and **CSS** — designed to replicate the look and feel of a real college/university website. This project showcases clean UI design, structured page layouts, and consistent styling across multiple pages.

## ✨ Features

### 🏠 Home / Landing Page
- Hero section with college banner and highlights
- Quick navigation to key sections
- Clean and welcoming layout for visitors

### 👤 About Page
- College history, vision, and mission
- Information about the institution
- Structured layout with headings and sections

### 🎓 Programmes Page
- List of academic programmes offered
- Course details and department info
- Organized with cards or structured layout

### 📋 Admission Page
- Admission process and eligibility criteria
- Step-by-step guidance for prospective students
- Important dates and requirements

### 📝 Apply Page
- Application form for prospective students
- Form fields for personal and academic details
- Clean form design with proper structure

### 🏅 Placement Page
- Placement statistics and records
- Recruiting companies and success stories
- Student achievement highlights

### 🎭 Beyond Academics Page
- Clubs, events, and extracurricular activities
- Campus life highlights
- Student engagement and cultural events

### 👨‍🏫 Faculty Page
- Faculty profiles and department-wise listing
- Qualifications and areas of expertise
- Professional card-style layout

### 🧑‍🎓 Student Life Page
- Campus life, hostel, and facilities
- Student community and activities
- Photo/gallery-style sections

### 📞 Contact Page
- Contact details, address, and map section
- Email and phone information
- Enquiry form for visitors

## 📁 Folder Structure

```
Collage-Website-Replica/
│
├── index.html              # 🏠 Home / Landing page
├── style.css               # 🎨 Global / shared stylesheet
│
├── about.html              # 👤 About the college
├── about.css               # Styles for About page
│
├── admission.html          # 📋 Admission info
├── admission.css           # Styles for Admission page
│
├── apply.html              # 📝 Application form page
│
├── beyond-academics.html   # 🎭 Clubs & extracurriculars
├── beyond-academics.css    # Styles for Beyond Academics page
│
├── contact.html            # 📞 Contact page
│
├── faculty.html            # 👨‍🏫 Faculty listing
├── faculty.css             # Styles for Faculty page
│
├── placement.html          # 🏅 Placement records
├── placement.css           # Styles for Placement page
│
├── programme.html          # 🎓 Academic programmes
├── programme.css           # Styles for Programme page
│
├── student-life.html       # 🧑‍🎓 Student life & campus
└── student-life.css        # Styles for Student Life page
```

## 🔗 How It Works

### 🧭 Navigation Flow

```
index.html → about.html / admission.html / programme.html / faculty.html / contact.html
```

### 📝 Admission Flow

```
index.html → admission.html → apply.html
```

### 🎓 Academic Flow

```
index.html → programme.html → faculty.html
```

### 🎭 Campus Life Flow

```
index.html → student-life.html → beyond-academics.html
```

### 🏅 Placement Flow

```
index.html → placement.html
```
